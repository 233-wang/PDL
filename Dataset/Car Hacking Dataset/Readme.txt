Car Hacking Dataset is provided by Hacking and Countermeasure Research Lab (HCRL) of Korea University through recording CAN traffic in a real vehicle.
The dataset includes one normal dataset and four different attack datasets: DoS, Fuzzy, spoofing GEAR and spoofing RPM.

Since the IDs in four different attack datasets are all hexadecimal, we provide the processed "normal_16_id.csv" dataset with hexadecimal IDs.

Above four different attack datasets and one normal dataset with decimal ID can be download in https://ocslab.hksecurity.net/Datasets/car-hacking-dataset and get the detailed explanation.


汽车黑客数据集由韩国大学黑客与对策研究实验室（HCRL）通过记录真实车辆的 CAN 流量提供。
该数据集包括一个正常数据集和四个不同的攻击数据集： DoS、模糊、欺骗 GEAR 和欺骗 RPM。

由于四个不同攻击数据集中的 ID 均为十六进制，因此我们提供了经过处理的带有十六进制 ID 的 "normal_16_id.csv "数据集。

以上四个不同的攻击数据集和一个十进制 ID 的正常数据集可在 https://ocslab.hksecurity.net/Datasets/car-hacking-dataset 中下载并获得详细说明。
